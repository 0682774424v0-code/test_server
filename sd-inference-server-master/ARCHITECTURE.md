### 2. Backend - Server (server.py)

```python
# Receives download request
def on_message(request):
  if request['type'] == 'download':
    do_download(request, folder, id, callback)

# Process download
def do_download(request, folder, id, callback):
  â†“
  # Extract tokens and URL
  url = request['url']
  civitai_token = request.get('civitai_token')
  hf_token = request.get('hf_token')
  â†“
  # Call download manager
  download_manager.download_model(
    url=url,
    model_type=type,
    folder=folder,
    progress_callback=callback,
    civitai_token=civitai_token,
    hf_token=hf_token
  )
```

### 3. Download Manager (download_manager.py)


### 4. Download Methods

#### aria2c Method (Fast)
```
aria2c \
  --console-log-level=error \
  -c                          # continue incomplete
  -x 16                       # 16 max connections
  -k 1M                       # 1MB min chunk
  -s 16                       # split into 16 parts
  -d /output/dir \
  -o filename \
  "https://download-url"
```

#### requests Method (Fallback)
```python
# Single-threaded streaming download
resp = requests.get(url, stream=True, headers=headers)
for chunk in resp.iter_content(chunk_size=8192):
    file.write(chunk)
    progress_callback(...)
```
